RakshaSetu

AI-Driven Disaster Prediction and Alert System

RakshaSetu is an AI-powered platform designed to predict, monitor, and alert users about potential disasters in real time. Unlike traditional news-based systems, RakshaSetu provides location-specific, live risk insights to enable faster and safer decision-making.

What It Solves:<br>
1. Delayed disaster information from news sources<br>
2. Lack of area-specific risk assessment<br>
3. Limited preparedness and early warning systems<br>

Key Features
1. AI-based disaster risk prediction<br>
2. Real-time, location-aware alerts<br>
3. Weather and environmental data analysis<br>
4. Emergency response guidance<br>
5. Low-latency alert dissemination<br>

Tech Stack<br>
Frontend: HTML, CSS, JavaScript<br>
Backend: Python (FastAPI / Flask)<br>
APIs: Weather and environmental data APIs, alert services<br>
Database: SQL / MySQL

How It Works:<br>
Live data collected → AI risk analysis → Threat level prediction → Alerts generated → User notified in real time.

Impact:
Improves disaster preparedness, reduces response time, and helps protect lives through proactive alerts.
